>>>> Comandos executar como admin
->Necessário registar pessoa com as seguintes credencias:
	- numero de cc: 1
	- password: admin

>>>> Necessário alterar o ip no código no RMIServer, AdminConsole e HeyBean


>>>> De forma a poder executar os ficheiros jar, executar o comando "java -jar [nome_ficheiro].jar" no terminal
na pasta que contém os ficheiros.

>>>> Para executar o ficheiro War.war, correr o servidor Tomcat no Intellij IDE, este será redirecionado para uma página Web.
